Currently, there are two components in 11be dApp:

  • Bolt: Wallet App

    • A graphical front-end with custom extensions for the official Ethereum full node (geth), which we build from scratch and provides unique features such as conditional transaction batch queue, password manager, and built-in ERC20 and gas price estimation supports. Additionally, it facilitates a package release meta tracking for both on-chain and off-chain contents using smart contract.

  • ElevenBuckets Exchange (11be)

    • Fully decentralized ERC20 token marketplace, which will have three different tiers:

    BoutiqueIndividual token holder with larger holdings can utilize our shop contract to list and sell tokens for Ether.
    ConsignmentIndividuals with smaller token holdings can jointly open a shop using our Token Pool contract.
    BucketsToken wholesale contracts utilizing price governance model and on-chain clearinghouse, both operates by community
