We believe tokenization via programmable Blockchain has greater potential beyond “FinTech” applications such as payment or security.

While often inevitably being dual-purposed as store of value, many tokens are created as symbols or emblems of participation that not only grant access to the associated dApps, but also enable sophiscated economic game theory for decentralized, self-regulated system governance, which makes them an essential and innovative element in Blockchain-based applications.

It is also our belief that, unlike stocks of a company, tokens with dApp reaching its maturity would require stable market price to encourage token circulation that ties to dApp utilization, instead of merely possessing them for financial gain.

Inspired by existing fully on-chain exchanges such as Kyber Network and Bancor, we introduce an alternative implementation with two new concepts:

I. discrete dynamic price range that allows near real-time price update without order maker micro-managing the listing price;

II. incentivised on-chain clearhouse via smart contract, which encourages participation of the platform day-to-day operation by sharing profit with participants;

It is our current plan to build 11be on top of Ethereum blockchain.

Many dApps on Ethereum network are hosted the same way as conventional website and utilize software extensions such as MetaMask for transactions in favor of easy access through web browsers.

While this approach certainly makes Blockchain and dApps more approachable to general public as well as web app developers, it discards most of the benefits of Blockchain which enables peer-to-peer network, rapid data queries via local block storage, and independent transaction verification.

In addition, the deep tie of dApps to web browser via conventional web hosting services make wallets and the associated private keys subject to malicious attacks and censorship.

11be is built with dedicated UI layer functioning like standalone apps instead of web browser, along with package management via on-chain metadata for verifications.


Currently, there are two components in 11be dApp:

  • Bolt: Wallet App

    • A graphical front-end with custom extensions for the official Ethereum full node (geth), which we build from scratch and provides unique features such as conditional transaction batch queue, password manager, and built-in ERC20 and gas price estimation supports. Additionally, it facilitates a package release meta tracking for both on-chain and off-chain contents using smart contract.

  • ElevenBuckets Exchange (11be)

    • Fully decentralized ERC20 token marketplace, which will have three different tiers:

    Boutique Individual token holder with larger holdings can utilize our shop contract to list and sell tokens for Ether.
    Consignment Individuals with smaller token holdings can jointly open a token consignment store using our Token Pool contract.
    Buckets Token wholesale contracts utilizing price governance model and on-chain clearinghouse, both operates by community
