Many dApps on Ethereum network are hosted the same way as conventional website and utilize software extensions such as MetaMask for transactions in favor of easy access through web browsers.

While this approach certainly makes Blockchain and dApps more approachable to general public as well as web app developers, it discards most of the benefits of Blockchain which enables peer-to-peer network, rapid data queries via local block storage, and independent transaction verification. In addition, the deep tie of dApps to web browser via conventional web hosting services make wallets and the associated private keys subject to malicious attacks and censorship.

11be is built with dedicated UI layer functioning like standalone apps instead of web browser, along with package management via on-chain metadata for verifications.
